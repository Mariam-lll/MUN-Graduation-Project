# MUN-Graduation-Project
**Contians 3 files:**
## index.html 
A simple two-tool web application called "Smart Utility Hub" built using HTML, CSS, and JavaScript
Tools:
### Password Validator
- Checks password strength and give a structured feedback BMI Calculator
- Calculates body mass index (BMI)
### UI design
- Dark themed interface
- Card for each tool with blurred background
- Hover animations

## password_tool.py
A command-line python tool that either generates passwords or check the strength of a given password. 
Tools:
### Password Generator
- Lets the user choose the length of the password
- Allows selection of character types of the password
### Password Strength Checker
- Checks for different elements of the password
- Provides clear feedback on missing elements

## Task3.py
A command-line python application that analyzes text files
Tools:
### Word and line counter
- Counts the total number of words and lines
### Text frequency checker
- Checks the frequency of a certain text
### File reporter
- Displays the total word count, line count, and number of characters
### File preview
- Prints the entire content of the file
### Last modification date
- Displays the last modification date of the file in a readable format
